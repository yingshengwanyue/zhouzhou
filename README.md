# 日记本 (ZhouZhou Diary)

一个支持多平台的个人日记本应用，用户可以方便地书写、编辑、查看和管理自己的日常日记。支持文字、图片等内容，具备本地存储和隐私保护功能，帮助用户以简洁直观的方式记录个人生活轨迹。

## ✨ 功能特点

- 📝 **简洁优雅的界面** - 白色主题，高级感设计，丝滑动画效果
- 🔐 **隐私保护** - 用户认证系统，密码加密存储
- 📸 **多媒体支持** - 支持在日记中添加图片（最多5张）
- 🔍 **快速搜索** - 实时搜索日记标题和内容
- 💾 **本地存储** - 使用SQLite数据库本地存储数据
- 📱 **响应式设计** - 支持桌面端、平板和移动端
- ⚡ **流畅动画** - 所有交互都有精心设计的动画效果

## 🚀 快速开始

### 环境要求

- Node.js 14.0 或更高版本
- npm 6.0 或更高版本

### 安装步骤

1. 克隆仓库
```bash
git clone https://github.com/yingshengwanyue/zhouzhou.git
cd zhouzhou
```

2. 安装依赖
```bash
npm install
```

3. 启动应用
```bash
npm start
```

4. 打开浏览器访问
```
http://localhost:3000
```

### 默认账户

- 用户名: `admin`
- 密码: `admin123`

## 📖 使用说明

### 登录
1. 在登录页面输入用户名和密码
2. 点击"登录"按钮进入主页面

### 创建日记
1. 点击左侧"新建日记"按钮
2. 输入日记标题和内容
3. 可选：点击"📷 添加图片"上传图片（支持 JPEG, JPG, PNG, GIF）
4. 点击"保存"按钮

### 查看日记
1. 在主页面点击任意日记卡片
2. 在弹出的窗口中查看完整内容
3. 点击图片可以放大查看

### 编辑日记
1. 在查看日记的窗口中，点击右上角的"✏️"编辑按钮
2. 修改标题、内容或添加新图片
3. 点击"保存"按钮

### 删除日记
1. 在查看日记的窗口中，点击右上角的"🗑️"删除按钮
2. 确认删除操作

### 搜索日记
1. 在顶部搜索框中输入关键词
2. 系统会实时显示匹配的日记

## 🛠️ 技术栈

### 后端
- **Node.js** - 运行环境
- **Express** - Web 框架
- **SQLite3** - 数据库
- **Multer** - 文件上传处理
- **bcryptjs** - 密码加密
- **express-session** - 会话管理

### 前端
- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **JavaScript (ES6+)** - 交互逻辑

## 🔒 安全特性

- 密码使用 bcrypt 加密存储
- 会话管理防止未授权访问
- 文件上传大小限制（5MB）
- 仅允许图片格式上传
- SQL 注入防护（使用参数化查询）
- XSS 攻击防护（HTML 转义）

## 📝 API 接口

### 认证接口
- `POST /api/login` - 用户登录
- `POST /api/logout` - 用户登出

### 日记接口
- `GET /api/diaries` - 获取所有日记
- `GET /api/diaries/:id` - 获取单篇日记
- `POST /api/diaries` - 创建新日记
- `PUT /api/diaries/:id` - 更新日记
- `DELETE /api/diaries/:id` - 删除日记
- `GET /api/diaries/search/:query` - 搜索日记
